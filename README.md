# DeepScan

DeepScan is a web-based deepfake detection system that uses deep learning techniques to identify manipulated images and videos.

## Features
- Image deepfake detection
- Video deepfake detection
- Confidence score prediction
- Web-based interface

## Technologies Used
- Python
- Flask
- TensorFlow
- OpenCV
- HTML/CSS
